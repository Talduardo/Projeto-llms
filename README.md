# 📑 Análise de Documentos Contábeis e Financeiros com LLMs

Este projeto aplica **Modelos de Linguagem (LLMs)** para otimizar processos de supervisão em setores contábeis e financeiros. A solução automatiza a criação de relatórios e reduz o tempo de leitura e interpretação de documentos complexos, tornando o acesso às informações mais rápido e eficiente.

---

## 🎯 Objetivos

- Automatizar a geração de relatórios contábeis e financeiros.  
- Reduzir o tempo de análise de documentos extensos.  
- Facilitar o acesso às informações mais relevantes por meio de resumos inteligentes.  

---

## 🧩 Metodologia

1. **Definição de Prompts** → Criação de diferentes estilos de prompts para guiar o modelo.  
2. **Aplicação nos Dados** → Testes realizados com relatórios contábeis do sistema **WEBGEX**.  
3. **Comparação dos Resultados** → Avaliação estatística da qualidade e completude dos resumos gerados.  

---

## 📂 Estrutura do Projeto

- **README.md** → Documentação do projeto  
- **llm_openai.py** → Implementação da API da OpenAI para geração de resumos  
- **llm's.ipynb** → Notebook Jupyter para configuração e testes com LLMs  
- **prompts.py** → Definição e organização dos prompts utilizados  
- **consideracoes.ipynb / consideracoes.md** → Reflexões e objetivos da análise  
- **relatorio.md** → Modelo de relatório criado a partir dos resultados
