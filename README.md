## Objetivos do código:
- Otimizar processos nos setores contabéis e financeiros;
- Integrar modelos de linguagem para automatizar a criação de relatórios;
- Reduzir o tempo de leitura e interpretação de documentos contábeis complexos, facilitando o acesso às informações mais relevantes.

###  Metodologia:
1. Definição de Prompts: Elaboração de dois estilos distintos de prompt para o modelo
2. Aplicação nos Dados: Testes com relatórios contábeis do sistema WEBGEX
3. Comparação dos Resultados: Uso de métricas para avaliar a clareza e completude dos resumos

- Código em python utilizando a API da OpenAI;
